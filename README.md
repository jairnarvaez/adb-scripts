# Hyprland Android Automation Scripts

Collection of scripts to control your Android device directly from Hyprland using keyboard shortcuts. Integrates ADB and scrcpy for seamless phone interaction without touching your device.

## Features

- **Launch apps on your phone** with keyboard shortcuts
- **Send files to WhatsApp contacts** directly from file manager
- **Open specific folders** in your phone's file explorer
- **Control your phone entirely from your keyboard**
